## PROMPT LANDING PAGE (STYLE WONDERFUL INDONESIA – CATEGORY PAGE)

> **Role**
> Kamu adalah **Senior Frontend Engineer + UI/UX Designer** dengan pengalaman membuat landing page pariwisata skala nasional.
>
> **Objective**
> Buat sebuah **landing page kategori destinasi** dengan **tampilan, struktur, dan pengalaman pengguna yang menyerupai** halaman kategori “Sumatera” di website Wonderful Indonesia.
>
> ---
>
> ### 1. Konsep & Visual
>
> * Gaya visual: **tourism / cultural / nature**
> * Look & feel: **clean, modern, immersive**
> * Dominasi **foto besar berkualitas tinggi**
> * Warna netral + aksen warna hangat
> * Typography elegan & mudah dibaca
>
> ---
>
> ### 2. Struktur Halaman (WAJIB ADA)
>
> #### A. Hero Section
>
> * Full-width hero banner
> * Background image/video destinasi
> * Overlay gradient tipis
> * Judul besar (contoh: *Explore Sumatera*)
> * Deskripsi singkat (1–2 kalimat)
> * Breadcrumb navigation
>
> ---
>
> #### B. Intro Section
>
> * Paragraf pengantar tentang wilayah
> * Max width (tidak full layar)
> * Fokus storytelling & atmosfer
>
> ---
>
> #### C. Content Grid (Main Section)
>
> * Grid card destinasi / artikel
> * Tiap card berisi:
>
>   * Featured image
>   * Category tag
>   * Judul
>   * Short description
> * Hover effect:
>
>   * Zoom image
>   * Shadow / overlay
> * Click → menuju detail page
>
> ---
>
> #### D. Filter & Navigation
>
> * Filter berdasarkan:
>
>   * Tema (alam, budaya, kuliner, event)
>   * Provinsi
> * Pagination atau “Load More”
>
> ---
>
> #### E. Highlight / Featured Section
>
> * Section khusus konten unggulan
> * Layout berbeda dari grid utama
> * Visual lebih besar
>
> ---
>
> #### F. Footer
>
> * Navigasi tambahan
> * Social media
> * Copyright
>
> ---
>
> ### 3. UX & Interaction
>
> * Smooth scrolling
> * Subtle animation (fade, slide)
> * Lazy loading image
> * Responsive:
>
>   * Desktop (grid 3–4 kolom)
>   * Tablet (2 kolom)
>   * Mobile (1 kolom)
>
> ---
>
> ### 4. Teknologi (Implementasikan)
>
> * Framework: **Next.js / React**
> * Styling: **Tailwind CSS**
> * Component-based structure
> * SEO-ready:
>
>   * Semantic HTML
>   * Meta title & description
> * Dummy data berbentuk JSON
>
> ---
>
> ### 5. Output yang Diharapkan
>
> * Kode landing page siap pakai
> * Reusable components:
>
>   * Hero
>   * Card
>   * Filter
> * Contoh dummy content
> * Clean & readable code
>
> ---
>
> **Rules**
>
> * Jangan copy konten asli
> * Fokus pada **struktur, UX, dan visual similarity**
> * Gunakan konten placeholder
>
> **Mulai dengan:**
>
> 1. Breakdown section
> 2. Wireframe singkat
> 3. Implementasi kode
