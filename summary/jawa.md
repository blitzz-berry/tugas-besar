## PROMPT LANDING PAGE (STYLE WONDERFUL INDONESIA – JAWA)

> **Role**
> Kamu adalah **Senior UI/UX Designer + Frontend Developer (Next.js/React)** dengan spesialisasi **tourism & destination landing pages**.
>
> ---
>
> **Objective**
> Buat sebuah **landing page destinasi wisata** dengan **tampilan, struktur, dan user experience menyerupai** halaman:
> *Wonderful Indonesia – Kategori Jawa*
>
> Landing page ini bertujuan untuk **promosi wilayah/destinasi**, bersifat **visual-first**, informatif, dan SEO-friendly.
>
> ---
>
> ### 1. Visual Style & Mood
>
> * Tema: **Cultural, Nature, Authentic Indonesia**
> * Warna dominan: **Putih, hitam, hijau alam, aksen emas/biru**
> * Typography:
>
>   * Heading: serif / elegant tourism font
>   * Body: clean sans-serif
> * Full-width layout, banyak **visual besar (hero images)**
> * Smooth scroll & subtle animations
>
> ---
>
> ### 2. Struktur Halaman
>
> #### A. Hero Section
>
> * Full-width banner image / slider
> * Judul besar: *“Explore Java”* (atau judul dinamis)
> * Subjudul singkat deskriptif
> * Overlay gradient
> * Breadcrumb navigation
>
> ---
>
> #### B. Introduction Section
>
> * Paragraf singkat tentang wilayah (Jawa)
> * Maks 2–3 paragraf
> * Fokus storytelling & cultural richness
>
> ---
>
> #### C. Highlight Content Grid
>
> * Grid artikel/destinasi (card-based layout)
> * Setiap card berisi:
>
>   * Featured image
>   * Category label
>   * Judul
>   * Short description
>   * Hover effect
> * Responsive (4 kolom desktop, 2 tablet, 1 mobile)
>
> ---
>
> #### D. Filter / Category Section
>
> * Filter konten berdasarkan:
>
>   * Alam
>   * Budaya
>   * Kuliner
>   * Event
> * Smooth transition saat filter berubah
>
> ---
>
> #### E. Featured Story / Editorial Section
>
> * Satu konten unggulan (large card)
> * Layout image + text (asymmetric)
> * CTA: “Read More”
>
> ---
>
> #### F. Pagination / Load More
>
> * Tombol “Load More” atau pagination
> * Lazy loading konten
>
> ---
>
> #### G. Footer
>
> * Navigasi tambahan
> * Social media icons
> * Copyright
>
> ---
>
> ### 3. Interaksi & UX
>
> * Hover animation pada cards
> * Smooth page transition
> * Lazy-loaded images
> * Scroll-based animation (fade-in, slide-up)
>
> ---
>
> ### 4. Teknologi
>
> * **Framework**: Next.js / React
> * **Styling**: Tailwind CSS / SCSS
> * **Component-based architecture**
> * Data konten dari **JSON / API dummy**
>
> ---
>
> ### 5. SEO & Accessibility
>
> * Semantic HTML5
> * Meta title & description
> * ALT text pada gambar
> * Proper heading hierarchy
>
> ---
>
> ### 6. Output yang Diharapkan
>
> * Landing page siap production
> * Responsive (desktop, tablet, mobile)
> * Clean & reusable components
> * Dummy content menyerupai struktur aslinya
>
> ---
>
> **Rules**
>
> * Jangan copy konten literal
> * Fokus pada **struktur, layout, dan experience**
> * Gunakan konten placeholder yang relevan
>

>
> 1. Wireframe singkat
