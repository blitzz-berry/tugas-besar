Siap. Ini **prompt yang fokus & tajam** untuk **membuat landing page yang mirip halaman kategori Sulawesi di Wonderful Indonesia**, tapi tetap **aman untuk edukasi / pengembangan**.

---

## PROMPT PEMBUATAN LANDING PAGE (WONDERFUL INDONESIA – CATEGORY STYLE)

> **Role**
> Kamu adalah **Senior Frontend Engineer + UI/UX Designer + Web Content Architect**.
>
> **Objective**
> Buat sebuah **landing page pariwisata** yang **menyerupai tampilan, struktur, dan pengalaman pengguna** dari halaman kategori:
> `wonderfulindonesia.co.id/category/sulawesi/`
>
> Landing page ini bertujuan sebagai **referensi desain & pengembangan internal**, bukan untuk menyalin konten resmi secara ilegal.
>
> ---
>
> ## 1. Desain & UI/UX
>
> * Gaya visual **modern, clean, editorial tourism website**
> * Dominasi **foto besar, immersive hero image**
> * Tipografi elegan & readable
> * Warna natural (biru laut, hijau alam, putih)
> * Responsive sempurna (desktop, tablet, mobile)
>
> Sertakan:
>
> * Hero section dengan judul kategori (contoh: *Explore Sulawesi*)
> * Subjudul deskriptif
> * Breadcrumb navigation
> * Smooth scrolling & hover animation
>
> ---
>
> ## 2. Struktur Halaman
>
> ### A. Hero Section
>
> * Background image fullscreen
> * Overlay gradient
> * Judul kategori
> * Deskripsi singkat wilayah
>
> ---
>
> ### B. Content Listing Section
>
> * Grid / masonry layout artikel
> * Setiap card berisi:
>
>   * Featured image
>   * Judul artikel
>   * Excerpt singkat
>   * Tag / kategori
>   * Tombol "Read More"
>
> ---
>
> ### C. Pagination / Load More
>
> * Tombol "Load More" atau pagination modern
>
> ---
>
> ### D. Sidebar (Opsional)
>
> * Search
> * Filter kategori
> * Popular posts
> * Tag cloud
>
> ---
>
> ### E. Footer
>
> * Navigasi tambahan
> * Social media icons
> * Copyright
>
> ---
>
> ## 3. Interaksi & Fitur
>
> * Hover effect pada card
> * Lazy loading image
> * Smooth transition antar section
> * SEO-friendly structure (H1–H3)
> * Accessible (alt image, contrast, semantic HTML)
>
> ---
>
> ## 4. Teknologi (Implementasikan Salah Satu)
>
> **Frontend Only (Landing Page):**
>
> * HTML5 + CSS3 + JavaScript
> * atau React / Next.js
> * Styling: Tailwind CSS / SCSS
>
> ---
>
> ## 5. Konten
>
> * Gunakan **dummy content** bertema Sulawesi:
>
>   * Alam
>   * Budaya
>   * Kuliner
>   * Destinasi wisata
> * Jangan gunakan teks asli dari website sumber
>
> ---
>
> ## 6. Output yang Diharapkan
>
> * Struktur folder project
> * Kode landing page lengkap
> * Komponen reusable (Card, Hero, Pagination)
> * Contoh data (JSON)
> * Dokumentasi singkat
>
> ---
>
> **Guidelines**
>
> * Fokus pada *look & feel*, bukan copy-paste konten
> * UI harus terasa seperti website pariwisata nasional
> * Siap dikembangkan ke CMS di tahap berikutnya
>
> **Mulai dengan:**
>
> 1. Wireframe layout
> 2. Breakdown section
> 3. Implementasi kode
