⚠️ **Catatan penting**: prompt ini **untuk tujuan edukasi, pembelajaran, riset UI/UX, atau pengembangan internal**. Website tersebut dilindungi hak cipta—jangan dipakai untuk publikasi komersial tanpa izin resmi.

---

## PROMPT CLONING WEBSITE (FULL FEATURE & CONTENT)

> **Role & Objective**
> Kamu adalah **Senior Full-Stack Web Engineer + UI/UX Analyst + Content Scraper Specialist**.
> Tugasmu adalah **mereplikasi (clone) website [https://www.wonderfullindonesia.co.id](https://www.wonderfullindonesia.co.id) secara lengkap**, dengan struktur, tampilan, konten, dan fitur yang semirip mungkin untuk **keperluan pembelajaran dan pengembangan internal**.
>
> ---
>
> **Scope Pekerjaan**
>
> ### 1. Visual & UI/UX
>
> * Replikasi **layout, grid system, spacing, typography, warna, ikon, animasi, dan transisi**
> * Responsive design **desktop, tablet, dan mobile**
> * Sticky navbar, dropdown menu, mega menu (jika ada)
> * Hero section dengan slider / video background
> * Hover effect, scroll animation, lazy loading
>
> ---
>
> ### 2. Struktur Halaman
>
> Clone **seluruh halaman**, termasuk tetapi tidak terbatas pada:
>
> * Home
> * Destinasi
> * Event
> * Kuliner
> * Budaya
> * Alam
> * Artikel / News / Blog
> * Galeri Foto & Video
> * About / Tentang
> * Kontak
> * Halaman detail (detail artikel, detail destinasi, dll)
>
> Pastikan **URL structure dan routing** konsisten dengan website asli.
>
> ---
>
> ### 3. Konten
>
> * Scrape & tampilkan **seluruh teks, gambar, video, metadata, judul, deskripsi**
> * Gunakan **CMS-friendly structure** (misal JSON / database schema)
> * Preserve:
>
>   * Heading hierarchy (H1–H6)
>   * SEO meta title & description
>   * ALT text gambar
>
> ---
>
> ### 4. Fitur & Interaksi
>
> Implementasikan seluruh fitur interaktif, termasuk:
>
> * Search (global & per kategori)
> * Filter & sorting konten
> * Pagination / infinite scroll
> * Carousel / slider
> * Video player
> * Form kontak
> * Social media integration
> * Map (jika ada)
> * Newsletter / subscription (dummy backend)
>
> ---
>
> ### 5. Teknologi (Sarankan & Implementasikan)
>
> Gunakan stack modern:
>
> * **Frontend**: React / Next.js + Tailwind / SCSS
> * **Backend**: Node.js / Express / Laravel (pilih salah satu)
> * **Database**: PostgreSQL / MySQL / MongoDB
> * **CMS-like structure** untuk konten
> * **API-ready architecture**
>
> Sertakan:
>
> * Folder structure
> * Component-based architecture
> * Reusable UI components
>
> ---
>
> ### 6. Performance & Best Practice
>
> * Optimasi image (lazy loading, responsive image)
> * SEO-friendly
> * Accessibility (ARIA, contrast, keyboard navigation)
> * Clean & maintainable code
>
> ---
>
> ### 7. Output yang Diharapkan
>
> * Source code lengkap
> * Struktur database / schema
> * Contoh data konten
> * Dokumentasi setup & running project
> * Catatan bagian yang **tidak boleh digunakan untuk publik**
>
> ---
>
> **Important Rules**
>
> * Jangan mengubah konten kecuali diminta
> * Fokus pada akurasi visual & fungsional
> * Asumsikan ini untuk **edukasi & riset internal**
>
> **Mulai dengan:**
>
> 1. Analisis website
> 2. Breakdown halaman & fitur
> 3. Rekomendasi tech stack
> 4. Implementasi bertahap
